# Crider's Institute of Welding Technology Website

This repository contains the marketing site for Crider's Institute of Welding Technology, built with Next.js and Tailwind CSS.

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.
